Gated Reccurent Unit (GRU)¶

In the context of recurrent neural networks (RNNs), the cell state is a memory unit that stores information over time. However, in the case of gated recurrent units (GRUs), the cell state is eliminated and the hidden state is used to transfer information across time steps.

A GRU consists of two gates: a reset gate and an update gate. The reset gate determines which information from the previous time step should be discarded and which information should be passed through to the current time step. The update gate determines how much information from the current time step should be combined with the information passed through from the previous time step.
