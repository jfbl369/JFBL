Fear&Greed + DAX visualization¶

I wrote a code to fetch the values of the Fear & Greed Index from an API and also the prices of the DAX (German stock index). My goal was to merge the two dataframes and display both datasets in the same graph.

To accomplish this, I used a Python library like Pandas to read the data from the API and the file containing DAX prices. Then, I merged the two dataframes using a merge function, most likely using the date column as the key.

Once the dataframes were merged, I used the Matplotlib library to create a graph. I plotted the Fear & Greed Index values and the DAX prices on the same graph, using separate axes for each dataset. To visually distinguish the two series, I used different colors for the bars representing the Fear & Greed Index and the lines representing the DAX prices.

The final result is a clear and informative graph showing the variations of the Fear & Greed Index and the DAX prices over time, allowing for analysis and comparison of the two datasets.
